# Get Started
Tucker Island is a Discord server in support of Matt, Mikayla, and Tucker. The server is acceptance-based and all members are friends & family.
<br/>
The server is home to a variety of fun features that have been fine-tuned to appeal to everyone. This page contains documentation on every feature and will be updated according to feedback, updates, and new features.
=== ":lucide-info: Vefora"
    !!! info inline end "Redirect"
        For more information you can visit the tab designated for our<br/>[Vefora bot][vefora].
    Vefora is a bot created by <span class="mention">@xyb</span> specifically for logging information that takes place in the server. It has documentation on all statistics relating to users, moderative actions, and can be especially helpful in court.
=== ":lucide-hand-coins: Economy"
    !!! info inline end "Redirect"
        For more information you can visit the tab designated for our<br/>[Economy system][eco].
    The economy system is built on [UnbelievaBoat][unb], a Discord bot marketed specifically for its economy commands and overall system. It's simple in function, and has numerous gambling games you can play to earn yourself more money.
=== ":lucide-scale: Court"
    !!! info inline end "Redirect"
        For more information you can visit the tab designated for our<br/>[Court system][court].
    Court is a dynamic and fun activity that engages members of [Tucker Island][disc]. It involves a judge, jury, prosecutor<span class="transparent">(s)</span>, defendant, and lawyer<span class="transparent">(s)</span>. The activity can generally span out to length a few hours, and involes both Criminal and Civil trials according to the server's list of Criminal Charges, and Suable Offenses.
